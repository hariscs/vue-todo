# 786 Todo App

Built with Vue3, Typescript, and Tailwind CSS.

## Environment Setup

Update the variables in `.env` according to your local setup. see `.env.example`

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
